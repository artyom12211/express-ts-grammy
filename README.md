#Get Started

## Create your bot.
Go to the @BotFather in telegram and create your bot.
Grab your bot token.

## Run this template
Copy this project via `git clone`
Go to the src directory and create `.env` file
Type in your `.env`:
PORT=3000
TGBOT_TOKEN=<YOUR_BOT_TOKEN>
Place your bot token instead <YOUR_BOT_TOKEN>
